Label image with its location
 
 Requester:   Chris Ward Reward:   $0.03 per HIT     Duration:   10 minutes
Qualifications Required:   Total approved HITs is not less than 200; HIT approval rate (%) is not less than 95

Label image with its location

In this task you will answer four questions about an image to determine the location of what's shown in the image.

Question 1: Is the image showing a location that's indoors or outdoors?

Any building or vehical interior is indoors, while anything that you could consider to be 'outside' is outdoors.

Select one answer for this question.

Question 2: Is the location in the image natural or man-made?

Natural locations are places that can be found in nature, or in the wild, while man-made locations have been constructed by humans.

Select one answer for this question.

Select an answer for either Question 3 or Question 4, but not both. If the location is man-made, select a function in Question 3 and don't select anything for Question 4. If the location is natural, select a natural location type in Question 4 and don't select anything for Question 3.

Question 3: If the location is man-made, what is its type or function?

Only select an answer for this question if you've selected 'man-made' for Question 2.

For each possible answer, here are some examples:

  • Transportation/urban: streets, sidewalks, city squares and plazas, car interiors, airports
  • Restaurant: bars, restaurants. Does not include kitchens in houses or apartments, which would be 'domestic'.
  • Recreation: athletic facilities, sports fields, dance clubs, concert venues, arcades, parks, gardens
  • Domestic: Interiors and exteriors of houses and apartments
  • Work/education: shops, office buildings, schools, farms, doctor's offices
  • Other/Unclear: any location that doesn't fit into the previous categories or if the location cannot be determined from the image

Remember to answer this question with respect to the location, not what the people in the picture are doing. For example, if the image shows children playing a game in the street or workers doing construction on a sidewalk, the correct answer would be 'transportation/urban', since streets and sidewalks are used for transportation.

Question 4: If the location in the picture is natural, what kind of natural location is it?

Only select an answer for this question if you've selected 'natural' for Question 2.

For each possible answer, here are some examples:

  • Body of water: lake, ocean, river, beach
  • Field: any (usually grassy) open field
  • Mountain: any part of a mountain or a similar elevated, usually rocky, location
  • Forest/jungle: Location covered with trees, undergrowth, or vegetation
  • Other/unclear: Any outside natural locations that don't fit into the previous categories or if the location cannot be determined from the image.

The following answer combinations will be rejected automatically:

  • Answers provided for both Question 3 (function) and Question 4 (landscape type)
  • 'indoors' selected for Question 1 and any answer provided for Question 4 (landscape type)
  • 'man-made' selected for Question 2 and any answer provided for Question 4 (landscape type)
  • 'natural' selected for Question 2 and any answer provided for Question 3 (function)
  • Question 3 and Question 4 both left unanswered (one of them needs to be answered)
